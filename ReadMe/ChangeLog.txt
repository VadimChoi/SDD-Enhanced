Версия 2.0

По сравнению с самой первой версией от @mahdiashtian, Эта версия умеет выполнять эти функции:
1. Создание папки Media и три под-папки Photo/Video/Voice и загрузка в эти папки в зависимости от типа
2. Игнорирование своих же отправленных медиа с самоуничтожением
3. Ведение логов в файл (и создание папки Logs) с ротацией в 5 МБ и лимитом файлов до 5
4. Загрузка по ссылке в двух вариантах t.me/c/<chat_id>/<message_id> и t.me/<channel_username>/<message_id>
5. Показ информации об отправителе (Юзернейм, номер телефона или ID)
6. Возможность загружать Голосовые сообщения с самоуничтожением
7. Подробное логирование ошибок
8. Игнорирование команд на загрузку по ссылке в других местах кроме личного ЛС
9. Папка со скриптами для быстрой очистки папки Logs и Media (для Windows, Linux и macOS)
10. Оптимизация кода под последнии версии Python и Telethon
